# dynamicTextCommunication

1. Clone the repo

   ```
   git clone https://github.com/DanyloPlotskykh/dynamicTextCommunication.git
   ```

2. Create folder build

   ```
   mkdir build
   ```

3. Compile

   ```
   make
   ```

4. Run the application

   ```
   #Linux
   ./build/server
   ./build/client
   ```
